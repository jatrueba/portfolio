CareCloud

Ruby on Rails Application

♦︎ Participated in CareCloud's Hackathon. Using the CareClouds API to create a checkin app that would reduce the time for patient check-in.

♦︎ Patients have the ability to update their demographics,insurance,upload documents using cell phone camera, pay copay/balance using an integrated credit card and get appointment reminders.

Movie Review App

Rails Application

♦︎ This is a Movie Review Application.

♦︎ You need to create an account to be able to add any movies to the Application.

♦︎ The movies come from the OMDB API.

♦︎ Used the gem searchkick in order to retrieve movies name, description, year, and picture from the API.

♦︎ You have the ability to give star ratings to each movie and add a review. The stars are added and the movie gets an average review from all the users. You're able to update your review or change it, but you may not do that to other people reviews.

♦︎

  • Date: July 2016
  • Project: Side Project
  • Category: Web Application
